# old_bestor

**Status: Deprecated. Do not extend or add templates to this theme.**

This is the previous version of the Bestor frontend theme, based on the `ruhi` parent theme. It has been superseded by `bestor` (child of `jakarta`).

The theme is retained in the repository to allow rollback if needed, but active development happens exclusively in `bestor`. The region set (`header_top_left`, `header_top_right`, `primary_menu`, `content_home_top`, etc.) is incompatible with the current `bestor`/`jakarta` region structure, so the two themes cannot be swapped without also updating all block placements.

No further documentation is provided for this theme.
